# Project Proposal: Penalty Tours

## What am I building? Who is it for?
I am building an interactive web-based penalty shootout game called Penalty Tours, where users can play in different stadium environments (field, indoor, street, beach). It is designed for students and casual users who want a simple and fun browser game.

## Why?
I wanted to build a project that is engaging and interactive while still being simple to use. This project solves the problem of static websites by creating a dynamic experience where users can actively play and see real-time results.

## MVP vs. Stretch Goals
**MVP:**
- Multiple pages (home, stadium selection, game, results)
- Basic penalty game 
- Score tracking during gameplay
- Responsive design for mobile and desktop

**Stretch Goals:**
- Save high scores using local storage
- Theme toggle (light/dark mode)
- Different visuals for each stadium
- Sound effects or animations

## Technologies
- HTML for structure
- CSS for styling and responsive design
- JavaScript for game logic, DOM manipulation, and event handling
- Local Storage for saving user data
