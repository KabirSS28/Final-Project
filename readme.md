# Penalty Tours

## Project Description
Penalty Tours is a simple web-based penalty shootout game. It is made for users who enjoy sports games and want a quick interactive football/soccer game in the browser. Users can choose a stadium, play as both the shooter and goalkeeper, and try to win the shootout.

## Live Demo URL
https://github.com/KabirSS28/Final-Project.git

## Features
- Choose from different stadiums: Field, Indoor Court, Street Football, and Beach
- Play as both the shooter and goalkeeper
- Click one of six goal zones to shoot or save
- Animated shooter, goalkeeper, and ball movement
- Scoreboard with penalty circles
- Green circles show goals and red circles show saves
- Sudden death mode if the game is tied after 5 rounds
- Dark mode effect during sudden death
- Final result popup with score and stats
- Confetti when the user wins

## AI Tools Used
I used ChatGPT to help plan the game structure, debug JavaScript issues, improve the CSS design, and make the game more responsive. It also helped with organizing the code and fixing problems like ball positioning, goalkeeper movement, team color switching, and stadium theme changes.

## Challenges Faced
One challenge was making the ball and goalkeeper move to the correct section of the goal. I solved this by adjusting the CSS positions for each goal zone. Another challenge was keeping the game screen clean and making it fit better on one screen, so I removed extra information boxes and kept only the important scoreboard and role display. I also had to fix the stadium themes so each stadium changed the background and goal color correctly.

## Future Improvements
With more time, I would add better player animations, more realistic sound effects, difficulty levels, and a tournament mode. I would also improve the mobile version even more and add a saved high score feature using local storage.
